
**SmartAlloc** is a desktop application for personal wealth management. Built with C# and WPF.

![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)
![Framework](https://img.shields.io/badge/.NET-8.0-purple.svg)

---

##  Key Features

-  **Zero-Knowledge Security:** Local SQLite database encrypted with **SQLCipher (AES-256)**. Your data never leaves your machine.
-  **Advanced Analytics:** Interactive dashboards with real-time spending charts powered by **LiveCharts2**.
-  **Live Currency Engine:** Real-time exchange rate integration (PLN/USD/EUR/CHF) via NBP API.
-  **Smart Goal Tracker:** Automated savings forecasting based on your 3-month spending velocity.
-  **Pro Reports:** Generate elegant, one-click monthly financial summaries in PDF format (**QuestPDF**).
-  **Modern GUI:** Full Dark Mode support, acrylic blur effects, and Material Design components.

---

##  Tech Stack

- **Language:** C# 12 / .NET 8
- **UI Framework:** WPF (Windows Presentation Foundation)
- **Design:** Material Design in XAML / Glassmorphism
- **Database:** SQLite + SQLCipher (Encrypted)
- **Libraries:**
  - `LiveCharts2` (Data Visualization)
  - `QuestPDF` (Report Generation)
  - `Dapper` (Lightweight ORM)
  - `Newtonsoft.Json` (API Handling)
